xdAnswers

Automatically select the correct answer on the page.

Highlight the correct option with a green border when UI is visible.

Ask AI to return only the answer — no explanation, no reasoning. Saves tokens.

Indicators: Overlay dot next to the correct answer.
Page title: Hide window. Answer shown only in browser tab title.
Stealth: Completely invisible. Answer auto-copied to clipboard.

Choose a preset theme or customize in Style tab.
